API name API description
1. Neighboring countries

Select a country from the drop down, and see a list of its neighboring countries.
2. Get Timezone

Input the latitude (between -90 and 90) and longitude (between -180 and 180) of your choice to see the timezone in GMT of that area.

3. Latest earthquake

Input values for north/south (between -90 and +90), and east/west (between -180 and +180). The result will show the most recent earthquake to happen in the area bounded by the values.
